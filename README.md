# 🛡️ PriVi-NetLens v5.2
**The Vanguard Command Center: Network Forensic Intelligence & IDS**

PriVi-NetLens is a high-performance, terminal-based Network Intrusion Detection System (IDS) and Forensic Analyzer. Designed by **PriViSecurity**, it bridges the gap between raw data and human understandability.



## 🔥 Key Features
* **Triple-Panel Dashboard:** A high-density UI providing real-time Protocol Stats, Device Intelligence, and a Forensic Packet Stream.
* **Passive OS Fingerprinting:** Instantly identifies **Windows, Linux, Android, and macOS/iOS** devices by analyzing TCP/IP stack signatures (TTL & Window Size).
* **Vanguard Security Audit:** Automatically flags unsecure, clear-text protocols like **HTTP, FTP, Telnet, and SMTP**.
* **Global Geo-IP Tracking:** Resolves external IP addresses to Country Codes with built-in rate-limiting for API stability.
* **Forensic Exporter:** Generates industry-standard `.pcap` files for Wireshark and human-readable `.txt` executive summaries.



## 🚀 Quick Start

### 1. Prerequisites
Ensure you have Python 3.x installed. PriVi-NetLens requires **root/administrator privileges** to capture raw network traffic.
